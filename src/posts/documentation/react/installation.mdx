# Introduction

webeditors-react is a component library designed on top of the CodeMirror package. It simplifies the creation of web editor components 
that support various programming languages by managing the complexities of these configurations behind the scenes.

## Installation 

To use the package, you'll need to install it with one of the following commands, depending on whether you're using npm, yarn, or pnpm.

``` npm i webeditors-react```

## Editors

The library offers various editor components tailored for different programming languages, formats, and markup languages. Each 
component has its own dedicated page in the documentation.
